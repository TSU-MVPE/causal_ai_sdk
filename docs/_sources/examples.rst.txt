Examples
========

The SDK is used as an async context manager. All service calls go through
``client.kg``, ``client.cd``, or ``client.da``.

Knowledge Graph
---------------

Create a session, upload a knowledge graph from file, list and inspect it:

.. code-block:: python

   async with CausalAIClient(api_key="...", base_url="...") as client:
       session = await client.kg.init_session()
       kg = await client.kg.upload_kg_from_file(
           session_uuid=session["uuid"],
           file_path="graph.json",
           title="My Graph",
       )
       kg_list = await client.kg.list_kg(session["uuid"])
       kg_detail = await client.kg.get_kg(session["uuid"], kg["id"])
       # When done: await client.kg.delete_session(session["uuid"])

Causal Discovery (MultiCa)
--------------------------

Upload multiple datasets with ``upload_data_for_multica`` (or use ``get_upload_url``
for a presigned URL and upload yourself), then run MultiCa with optional roots/sinks.
Poll until complete and fetch the result:

.. code-block:: python

   async with CausalAIClient(api_key="...", base_url="...") as client:
       session = await client.kg.init_session()
       uploaded = await client.cd.upload_data_for_multica(
           session["uuid"], ["dataset1.csv", "dataset2.csv"]
       )
       task = await client.cd.run_multica(
           uploaded, threshold=0.01, roots=["X1"], sinks=["Y"]
       )
       status = await client.cd.wait_for_task(uploaded, timeout=300, interval=5)
       result = await client.cd.get_task_result(session["uuid"], task["task_id"])

Causal Discovery (LiNGAM)
-------------------------

Upload a single dataset, run LiNGAM causal discovery, wait for completion, and
fetch the result:

.. code-block:: python

   async with CausalAIClient(api_key="...", base_url="...") as client:
       session = await client.kg.init_session()
       uploaded = await client.cd.upload_data_for_lingam(session["uuid"], "data.csv")
       task = await client.cd.run_lingam(uploaded, threshold=0.01)
       await client.cd.wait_for_task(uploaded, timeout=600, interval=5)
       task_result = await client.cd.get_task_result(session["uuid"], task["task_id"])
       # task_result contains result_url; fetch JSON from it for feature_names, etc.

Causal Discovery (TraCKR)
-------------------------

Upload a source knowledge graph, upload target data, run TraCKR matching, then
run TraCKR CD with knowledge transfer. Wait for completion and fetch the result:

.. code-block:: python

   async with CausalAIClient(api_key="...", base_url="...") as client:
       session = await client.kg.init_session()
       kg = await client.kg.upload_kg_from_file(
           session_uuid=session["uuid"], file_path="source_graph.json"
       )
       uploaded = await client.cd.upload_data_for_trackr(session["uuid"], "target.csv")
       await client.cd.start_trackr_matching(uploaded, source_kg_id=kg["id"])
       matching = await client.cd.wait_for_matching(
           session["uuid"], mode="trackr", timeout=300, interval=3
       )
       # Optionally adjust: await client.cd.set_trackr_matching(session["uuid"], {...})
       task = await client.cd.run_trackr(
           uploaded,
           transferred_knowledge={"session_uuid": session["uuid"], "kg_id": kg["id"]},
           threshold=0.01,
       )
       await client.cd.wait_for_task(uploaded, timeout=300, interval=5)
       result = await client.cd.get_task_result(session["uuid"], task["task_id"])

Decision Analysis
-----------------

Run explain or enumerate using a CD result reference. Poll until complete and
fetch the result:

.. code-block:: python

   async with CausalAIClient(api_key="...", base_url="...") as client:
       session = await client.kg.init_session()
       # cd_result_reference = session_uuid + task_id from a completed CD task
       explain_task = await client.da.run_explain(
           session_uuid=session["uuid"],
           cd_result_reference={"session_uuid": "...", "task_id": "..."},
           current_observation={"feature_a": 0.5, "feature_b": 1.0},
           targets=[{"col": "target_col", "sense": ">", "threshold": 0.8}],
       )
       await client.da.wait_for_task(
           session["uuid"], explain_task["task_id"], timeout=300, interval=5
       )
       da_result = await client.da.get_task_result(
           session["uuid"], explain_task["task_id"]
       )
       # Or run_enumerate with same args for enumeration:
       # enum_task = await client.da.run_enumerate(...)

Column matching (MultiCa)
~~~~~~~~~~~~~~~~~~~~~~~~~~

Upload data, start MultiCa column matching, wait for it to complete, then run MultiCa
using the same session (and optionally set or inspect the matching).

.. code-block:: python

   from causal_ai_sdk.models import UploadedData

   async with CausalAIClient(api_key="...", base_url="...") as client:
       session = await client.kg.init_session()
       upload_info = await client.cd.get_upload_url(session["uuid"])
       # ... upload your data to S3 using upload_info ...
       uploaded = UploadedData(
           task_id=upload_info.task_id,
           session_uuid=session["uuid"],
           s3_key=upload_info.s3_key,
       )
       await client.cd.start_multica_matching(uploaded)
       matching = await client.cd.wait_for_matching(session["uuid"], mode="multica")
       # Optionally: client.cd.set_multica_matching(session["uuid"], {"col_a": "src_a"})
       task = await client.cd.run_multica(uploaded, threshold=0.01)
