class causal_ai_sdk.client.CausalAIClient(api_key, base_url, http_client=None, config=None)[source]

Bases: object

Main client for Causal AI SDK.

This client provides a unified interface to access all services. It implements the Facade pattern, hiding the complexity of service initialization and configuration.

The client must be used as an async context manager. Do not use without async with CausalAIClient(...) as client:.

Example

>>> async with CausalAIClient(
...     api_key="your-api-key",
...     base_url="https://api.example.com"
... ) as client:
...     session = await client.kg.init_session()
...     kg = await client.kg.upload_kg_from_file(
...         session_uuid=session.uuid,
...         file_path="graph.json"
...     )
Parameters:

  • api_key (str)

  • base_url (str)

  • http_client (Optional[HTTPClient])

class causal_ai_sdk.config.Config(api_key=None, base_url=None, timeout=120, **kwargs)[source]

Bases: object

Configuration class for Causal AI SDK.

Manages API key, base URL, and other configuration settings. Supports reading from environment variables.

Parameters:

  • api_key (Optional[str]) – API Key for authentication. If not provided, will try to read from CAUSAL_AI_API_KEY environment variable.

  • base_url (Optional[str]) – Base URL of the API Gateway. If not provided, will try to read from CAUSAL_AI_BASE_URL environment variable.

  • timeout (int) – Request timeout in seconds. Defaults to 120.

  • **kwargs (Any) – Additional configuration options.

Raises:

ValueError – If required configuration is missing or invalid.

Example

>>> # From parameters
>>> config = Config(
...     api_key="your-api-key",
...     base_url="https://api.example.com"
... )
>>>
>>> # From environment variables
>>> # Set CAUSAL_AI_API_KEY and CAUSAL_AI_BASE_URL
>>> config = Config()
